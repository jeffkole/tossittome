How to catch

Pages are caught by a Chrome extension. Here's how to install it:

  1. Download the extension. (when Chrome objects, click "Ok").
  2. In Chrome, navigate to chrome://extensions.
  3. Drag "tossittome.crx" to the tab with the extensions page open.
  4. Grant the permissions.

Happy tossing.
