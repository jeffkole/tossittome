How to catch pages

Pages are caught by a Chrome extension. Here's how to install it:

  1. Download the extension: Extension. Click "ok" when Chrome tells you it refuses to install the extension.
  2. In Chrome, navigate to chrome://extensions
  3. Drag the extension "tossittome.crx" to the Chrome window showing the extension page.
  4. Approve the permissions.
